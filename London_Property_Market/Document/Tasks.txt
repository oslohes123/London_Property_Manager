Base Tasks

Creating the basic window
------------

Application Window

BorderPane
Top contains from selection label
Middle contains some pane with additional data
Bottom contains some buttons


------------

The welcome

BorderPane
Top contains from-to filter
Middle contains information on how the program works
-> Contains a labelled image of how it works
Bottom contains buttons

------------
The map window

Middle contains map of boroughs as buttons
A view button will show the listings in each of
the selected boroughs as well as the statistics of the borough(s)


------------
The statistics window

Will open up separately once the view button has been pressed
If multiple boroughs have been selected, the statistics would be recalculated
with each of the borough included so that only one set of statistics
are outputted
